Scrape-Detik.com adalah sebuah alat (*tool*) berbasis nodejs yang dirancang untuk mengambil data berita dari portal berita [Detik.com](https://www.detik.com) secara otomatis. Tool ini memudahkan pengguna untuk mengumpulkan data artikel untuk keperluan riset, analisis data, atau pembelajaran *machine learning*.

## âœ¨ Fitur Utama
- **Ekstraksi Berita**: Mengambil judul, ringkasan, tanggal publikasi, dan URL artikel.
- **Support Multi-page**: Mendukung pengambilan data dari banyak halaman hasil pencarian.
- **Ekspor Data**: Menyimpan hasil scraping ke format CSV atau JSON (sesuaikan dengan kode kamu).
